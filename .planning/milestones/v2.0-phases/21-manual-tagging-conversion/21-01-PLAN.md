  <action>
Add two new tables to schema.ts following existing patterns:

1. `tags` table:
   - id: uuid primary key defaultRandom
   - userId: uuid NOT NULL, references users.id with onDelete cascade
   - name: varchar(50) NOT NULL
   - color: varchar(7) NOT NULL (hex color like "#3B82F6")
   - createdAt: timestamp with timezone, defaultNow, NOT NULL
   - Add uniqueIndex on (userId, name) to prevent duplicate tag names per user

2. `transactionTags` junction table:
   - transactionId: uuid NOT NULL, references transactions.id with onDelete cascade
   - tagId: uuid NOT NULL, references tags.id with onDelete cascade
   - createdAt: timestamp with timezone, defaultNow, NOT NULL
   - primaryKey on (transactionId, tagId)
   - index on transactionId for fast lookups
   - index on tagId for fast lookups

3. Add relations:
   - users -> many tags
   - tags -> one user
   - tags -> many transactionTags
   - transactions -> many transactionTags
   - transactionTags -> one transaction, one tag

4. Add type exports:
   - Tag = typeof tags.$inferSelect
   - NewTag = typeof tags.$inferInsert
   - TransactionTag = typeof transactionTags.$inferSelect
   - NewTransactionTag = typeof transactionTags.$inferInsert

Follow the exact patterns from categories table for styling.
