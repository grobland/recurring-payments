Create ReimportWizard component:

- Maintain wizard state: { queue, currentIndex, completed }

For each transaction:

2. Pre-fill subscription form with transaction data:
   - name: merchantName
   - amount: transaction amount
   - currency: transaction currency
   - frequency: "monthly" (default)
   - nextRenewalDate: calculated from transaction date + 1 month
3. Show progress: "Creating subscription 2 of 5"
4. On form submit, call POST /api/transactions/[id]/convert
5. Advance to next transaction

UI structure:
```

[Progress bar]

[Subscription Form pre-filled]
[Skip] [Save & Continue]
```

Handle edge cases:
- Skip button: mark transaction as skipped (update tagStatus to not_subscription)
- Cancel wizard: confirm dialog if mid-process
- All complete: show success summary and close

Use existing SubscriptionForm component for the form (or create simplified inline version if SubscriptionForm is too complex).
