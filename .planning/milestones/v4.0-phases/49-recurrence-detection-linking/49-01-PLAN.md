**Key function: `detectRecurringSeries(db, userId): Promise<DetectionResult>`**

Implementation steps:
1. Load all user transactions with normalizedDescription and sourceHash (pipeline-created ones). IMPORTANT: transactions do NOT have a merchantEntityId column. Group by looking up each transaction's normalizedDescription in merchant_aliases to get the merchantEntityId. Use a single query joining transactions -> merchant_aliases on `lower(alias_text) = lower(normalized_description)` and `userId`.

2. Load user_transaction_labels. Exclude transactions labeled "not_recurring" or "ignore". For transactions labeled "recurring", add +0.10 confidence bonus later.

3. Skip merchant groups with < 2 transactions (DETECT-10).

4. For each merchant group, check if the merchantEntityId is already linked to a recurring_master via recurring_master_series_links. If yes: this is Rule A (alias hit). Confidence = 0.95.

5. If Rule A doesn't match, sort transactions by date ascending and compute intervals:
   - `intervals = [txn[i+1].date - txn[i].date for i in range(len-1)]` in days

6. Rule B (fixed monthly): Check if median interval is 25-35 days AND coefficient of variation of amounts <= 0.05 AND day-of-month stddev <= 5. Base confidence = 0.70. Bonus: +0.05 per occurrence beyond 2 (max +0.15), +0.05 if all amounts exactly equal.

7. Rule C (variable monthly): Check if median interval is 25-35 days AND CV of amounts > 0.05 but <= 0.50. Base confidence = 0.60. Bonus: +0.05 per occurrence beyond 2 (max +0.10), +0.05 if interval stddev < 3 days.

8. Rule D (annual): Check if median interval is 335-395 days. Base confidence = 0.65. Can be fixed or variable.

9. Rule E (weekly/quarterly/custom):
   - Weekly: median interval 5-9 days, min 3 occurrences. Confidence = 0.65.
   - Quarterly: median interval 80-100 days, min 2 occurrences. Confidence = 0.60.
   - Custom: if no standard cadence matches but intervals are consistent (CV of intervals < 0.3), use median interval. Confidence = 0.55.

10. First matching rule wins per merchant group.
