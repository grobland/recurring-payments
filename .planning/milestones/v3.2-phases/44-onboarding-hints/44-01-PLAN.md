**Helper functions (not exported, same pattern as overlap hook):**
```typescript
function readHintDismissals(): Record<string, boolean> {
  if (typeof window === "undefined") return {};
  try {
    const raw = localStorage.getItem(STORAGE_KEY);
    if (!raw) return {};
    return JSON.parse(raw) as Record<string, boolean>;
  } catch {
    return {};
  }
}

function writeHintDismissals(dismissals: Record<string, boolean>): void {
  if (typeof window === "undefined") return;
  try {
    localStorage.setItem(STORAGE_KEY, JSON.stringify(dismissals));
  } catch {
    // Silently ignore storage errors (private browsing, quota exceeded)
  }
}
```

**Hook:**
```typescript
export function useHintDismissals(): {
  isDismissed: (pageId: string) => boolean;
  dismiss: (pageId: string) => void;
}
```

- Use `useState<Record<string, boolean>>(() => readHintDismissals())` for lazy initialization
- `isDismissed(pageId)`: return `!!dismissals[pageId]`
- `dismiss(pageId)`: read current from localStorage, merge `{ [pageId]: true }`, write back, update React state with `setDismissals`

This is simpler than the overlap hook — no group signatures, no cleanup effect, no re-surface logic. Dismissal is permanent.

```typescript
export function DismissibleEmptyState({
  pageId,
  dismissedText,
  icon,
  title,
  description,
  primaryAction,
  secondaryAction,
}: DismissibleEmptyStateProps) {
  const { isDismissed, dismiss } = useHintDismissals();

  // Post-dismiss: minimal text line
  if (isDismissed(pageId)) {
    return (
      <p className="py-12 text-center text-sm text-muted-foreground">
        {dismissedText}
      </p>
    );
  }

  // Full empty state with dismiss X button
  return (
    <div className="relative">
      <button
        onClick={() => dismiss(pageId)}
        className="absolute right-2 top-2 rounded-sm p-1 text-muted-foreground/60 hover:text-muted-foreground transition-colors focus:outline-none focus-visible:ring-2 focus-visible:ring-ring"
        aria-label="Dismiss hint"
      >
        <X className="h-4 w-4" />
      </button>
      <EmptyState
        icon={icon}
        title={title}
        description={description}
        primaryAction={primaryAction}
        secondaryAction={secondaryAction}
      />
    </div>
  );
}
```

Import `X` from `lucide-react`, `EmptyState` from `@/components/shared/empty-state`, and `useHintDismissals` from `@/lib/hooks/use-hint-dismissals`.

Key design decisions:
- The `relative` wrapper + `absolute` X button positions the dismiss control at top-right of the empty state area (per CONTEXT.md decision)
- Post-dismiss state shows a simple centered text line that feels intentional, not broken (per CONTEXT.md: "a simple 'No subscriptions yet' line is better than an empty void")
- No animation on dismiss — instant hide (simplest approach, per Claude's discretion)
- The X button uses `text-muted-foreground/60` so it's visible but not visually dominant
