```typescript
export type OverlapGroup = {
  categoryId: string;
  subscriptionIds: string[];
};
```

Export a pure function (for testing):
```typescript
export function computeOverlapGroups(subscriptions: Array<{
  id: string;
  categoryId: string | null;
  status: string;
}>): Map<string, string[]>
```

Logic:
1. Filter to only subscriptions where `status === "active"` AND `categoryId !== null`
2. Group by `categoryId` using a Map
3. Return only entries where the group has 2+ members

Export a React hook:
```typescript
export function useOverlapGroups(subscriptions: Array<{...}>): Map<string, string[]>
```
Uses `useMemo` to memoize `computeOverlapGroups(subscriptions)` with `[subscriptions]` dependency.
