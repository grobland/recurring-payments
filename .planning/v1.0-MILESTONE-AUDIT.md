## E2E User Flows

### Flow 1: PDF Import
**Status:** COMPLETE

1. Upload PDF → dropzone in import/page.tsx
2. Process files → POST /api/import
3. AI extraction → OpenAI GPT-4 Vision via pdf-parser.ts
4. Review results → UI displays extracted subscriptions
5. Confirm import → POST /api/import/confirm
6. View subscriptions → router.push("/subscriptions")

### Flow 2: CRUD Operations
**Status:** COMPLETE (7 E2E tests passing)

2. View in list → /subscriptions page → GET /api/subscriptions
3. Edit subscription → /subscriptions/[id]/edit → PATCH /api/subscriptions/[id]
4. Delete subscription → Actions menu → DELETE /api/subscriptions/[id]
