2. **calculateVolatility(historicalMonthly: number[]): number**
   - Returns coefficient of variation (stdDev / mean)
   - Handle edge cases:
     - Less than 2 data points: return 0.20 (conservative 20% default)
     - Zero or negative mean: return 0.20
   - Cap at 1.0 (100% volatility max to prevent unreasonable bands)
   - Use sampleStandardDeviation and mean from simple-statistics

3. **scaleUncertainty(baseStdDev: number, monthsAhead: number): number**
   - Returns baseStdDev * Math.sqrt(monthsAhead)
   - Implements sqrt(time) scaling for independent monthly variations
   - This is why fan charts "fan out" over time

4. **addConfidenceIntervals(forecast: number, volatility: number, monthsAhead: number = 1)**
   - Returns object with { lower80, upper80, lower95, upper95 }
   - Calculate forecastStdDev = forecast * volatility
   - Scale by sqrt(monthsAhead) for future projections using scaleUncertainty
   - Use Math.max(0, ...) to floor all lower bounds at zero (spending cannot be negative)
   - Apply z-scores: 1.28 for 80%, 1.96 for 95%

     const today = new Date();
     const endDate = addDays(today, days);
     const upcomingCharges: UpcomingCharge[] = [];

     for (const sub of activeSubs) {
       let renewalDate = new Date(sub.nextRenewalDate);
       const incrementMonths = sub.frequency === 'monthly' ? 1 : 12;

       while (renewalDate <= endDate) {
         if (renewalDate >= today) {
           upcomingCharges.push({
             date: renewalDate.toISOString(),
             subscriptionId: sub.id,
             subscriptionName: sub.name,
             amount: parseFloat(sub.amount),
             currency: sub.currency,
             categoryId: sub.categoryId,
             categoryName: sub.categoryName || 'Uncategorized',
             categoryColor: sub.categoryColor || '#6b7280',
           });

       }
     }
