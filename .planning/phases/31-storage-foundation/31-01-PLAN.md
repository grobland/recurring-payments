1. Install the Supabase JS client: `npm install @supabase/supabase-js`

2. Create `src/lib/supabase/server.ts` — service-role Supabase client singleton:
   - Import `createClient` from `@supabase/supabase-js`
   - Create and export `supabaseAdmin` using `process.env.NEXT_PUBLIC_SUPABASE_URL` and `process.env.SUPABASE_SERVICE_ROLE_KEY`
   - Disable auth session management: `{ auth: { persistSession: false, autoRefreshToken: false, detectSessionInUrl: false } }`
   - CRITICAL: `SUPABASE_SERVICE_ROLE_KEY` must NOT have `NEXT_PUBLIC_` prefix — it must never reach the browser

3. Create `src/lib/storage/pdf-storage.ts` — PDF storage helpers:
   - `uploadStatementPdf(file: File, userId: string, sourceType: string): Promise<{ path: string } | null>`
     - Guard: return null if `file.size > 10 * 1024 * 1024` (10 MB)
     - Guard: return null if `file.type !== 'application/pdf'`
     - Build path: `{userId}/{yyyy-MM}/{sourceSlug-yyyy-MM}.pdf` using current date (statement date not yet known at upload time per research open question #3)
     - `sourceSlug` = `sourceType.toLowerCase().replace(/\s+/g, '-').replace(/[^a-z0-9-]/g, '')`
     - Convert file to Buffer: `Buffer.from(await file.arrayBuffer())`
     - Upload via `supabaseAdmin.storage.from('statements').upload(path, buffer, { contentType: 'application/pdf', upsert: true, cacheControl: '3600' })`
     - On error: `console.error('PDF storage upload failed:', error)` and return null (non-fatal)
     - On success: return `{ path }`
   - `generatePdfSignedUrl(pdfStoragePath: string): Promise<string | null>`
     - Call `supabaseAdmin.storage.from('statements').createSignedUrl(pdfStoragePath, 3600)` (1-hour expiry per Claude's discretion)
     - Return `data.signedUrl` on success, null on error

4. Create `src/app/api/statements/[id]/pdf-url/route.ts` — on-demand signed URL endpoint:
   - `GET` handler: authenticate with `auth()`, verify `session?.user?.id`
   - Fetch statement from DB: `db.query.statements.findFirst()` where `id` matches and `userId` matches (ownership check)
   - If no statement or no `pdfStoragePath`: return 404 `{ error: 'No PDF stored' }`
   - Call `generatePdfSignedUrl(statement.pdfStoragePath)`
   - Return `{ url: signedUrl }` on success, 500 on error

5. Update `.env.example` — add the three Supabase env vars at the end:
   ```
   # Supabase Storage
   NEXT_PUBLIC_SUPABASE_URL=""
   NEXT_PUBLIC_SUPABASE_ANON_KEY=""
   SUPABASE_SERVICE_ROLE_KEY=""
   ```
