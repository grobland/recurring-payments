<task type="checkpoint:human-action" gate="blocking">
  <name>Task 2: User creates API keys from service dashboards</name>
  <instructions>
You need to create API keys from three services. Open each link, create the key, and paste it into .env.local.

## OpenAI API Key

1. Go to https://platform.openai.com/api-keys
2. Sign up or log in
3. Click "Create new secret key"
4. Copy the key (starts with sk-proj-...)
5. Paste into .env.local as OPENAI_API_KEY value
6. IMPORTANT: Add payment method at https://platform.openai.com/account/billing (required for API usage)

## Stripe API Keys (TEST MODE)

1. Go to https://dashboard.stripe.com/test/apikeys
2. Sign up or log in
3. Make sure "Test mode" toggle is ON (orange badge visible)
4. Copy "Secret key" (starts with sk_test_...)
5. Paste into .env.local as STRIPE_SECRET_KEY value
6. Copy "Publishable key" (starts with pk_test_...)
7. Paste into .env.local as STRIPE_PUBLISHABLE_KEY value

## Resend API Key

1. Go to https://resend.com/api-keys
2. Sign up or log in
3. Click "Create API Key"
4. Give it a name (e.g., "Subscription Manager Dev")
5. Copy the key (starts with re_...)
6. Paste into .env.local as RESEND_API_KEY value
  </instructions>
  <resume-signal>Type "keys added" when all three API keys are in .env.local</resume-signal>
</task>
