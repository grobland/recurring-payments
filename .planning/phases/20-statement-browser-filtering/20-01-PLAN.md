Create GET endpoint at /api/transactions with:

**Authentication:** Require session via auth(), return 401 if not authenticated.

**Query Parameters:**
- sourceType: filter by statement.sourceType (requires join)
- tagStatus: filter by transactions.tagStatus (skip if "all")
- dateFrom: filter transactions >= date
- dateTo: filter transactions <= date
- search: ilike on merchantName OR categoryGuess
- cursorDate: cursor transactionDate (ISO string)
- cursorId: cursor id (UUID)

**Keyset Pagination Logic (from RESEARCH.md):**
1. Build conditions array starting with userId filter
2. Add optional filters based on query params
3. For cursor: use OR condition with (date < cursorDate) OR (date = cursorDate AND id < cursorId)
4. Order by transactionDate DESC, id DESC
5. Fetch PAGE_SIZE + 1 to detect hasMore
6. Slice to PAGE_SIZE, build nextCursor from last item

**Join with statements** to get sourceType:
```typescript
const results = await db
  .select({
    ...transactions,
    sourceType: statements.sourceType,
  })
  .from(transactions)
  .leftJoin(statements, eq(transactions.statementId, statements.id))
  .where(and(...conditions))
  .orderBy(desc(transactions.transactionDate), desc(transactions.id))
  .limit(PAGE_SIZE + 1)
```

**Response shape:**
```json
{
  "transactions": [...],
  "nextCursor": { "transactionDate": "...", "id": "..." } | null,
  "hasMore": boolean
}
```

**Hook Signature:**

```typescript
export function useTransactions(filters: TransactionFilters)
```

**Implementation:**
1. Import useInfiniteQuery from @tanstack/react-query
2. Import TransactionFilters, TransactionPage, TransactionCursor from types
3. Build query key: ['transactions', filters]
4. queryFn: build URLSearchParams from filters + pageParam cursor, fetch /api/transactions
5. initialPageParam: null (first page has no cursor)
6. getNextPageParam: return lastPage.nextCursor if hasMore, else undefined

**Query function pattern:**
```typescript
queryFn: async ({ pageParam }) => {
  const params = new URLSearchParams()
  if (filters.sourceType) params.set('sourceType', filters.sourceType)
  if (filters.tagStatus && filters.tagStatus !== 'all') params.set('tagStatus', filters.tagStatus)
  if (filters.dateFrom) params.set('dateFrom', filters.dateFrom)
  if (filters.dateTo) params.set('dateTo', filters.dateTo)
  if (filters.search) params.set('search', filters.search)
  if (pageParam) {
    params.set('cursorDate', pageParam.transactionDate)
    params.set('cursorId', pageParam.id)
  }

  const res = await fetch(`/api/transactions?${params}`)
  if (!res.ok) throw new Error('Failed to fetch transactions')
  return res.json() as Promise<TransactionPage>
}
```

**Return all useInfiniteQuery properties** plus computed helpers:

- totalLoaded: count of loaded transactions
