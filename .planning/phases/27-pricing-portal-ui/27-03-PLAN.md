<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Configure Stripe Customer Portal for tier switching</name>
  <action>
Configure the Stripe Customer Portal to allow users to switch between subscription tiers.

**Steps:**

1. Go to Stripe Dashboard -> Settings -> Billing -> Customer portal
   URL: https://dashboard.stripe.com/settings/billing/portal

2. In the "Features" section, enable:
   - [x] Customers can switch plans
   - Set proration behavior to "Credit back customers for time remaining" (for prorated billing)

3. In the "Products" section:
   - Click "Add products"
   - Add all three tier products: Primary, Enhanced, Advanced
   - For each product, ensure all price IDs (monthly and annual) are included
   - Maximum 10 products allowed in portal

4. Save changes

**What this enables:**
- Users clicking "Manage Billing" will see option to "Switch plan"
- Plan switches will be prorated (partial month credit applied to new plan)
- All three tiers visible in portal for users to choose
  </action>
  <verify>
1. Open a test user's billing portal (click "Manage Billing" from /settings/billing)
2. Verify "Switch plan" or "Change plan" option is visible
3. Verify all three tiers (Primary, Enhanced, Advanced) appear as options
4. Verify pricing shows correctly for each tier
  </verify>
  <resume-signal>
Type "portal configured" when tier switching is enabled in Stripe Dashboard
  </resume-signal>
</task>

<task type="checkpoint:human-action" gate="blocking">
  <name>Task 2: Configure Stripe portal branding</name>
  <action>
Add app branding to the Stripe Customer Portal for consistent user experience.

**Steps:**

1. Go to Stripe Dashboard -> Settings -> Branding
   URL: https://dashboard.stripe.com/settings/branding

2. Upload logo:
   - Icon: Square logo (at least 128x128px)
   - Logo: Full logo for portal header

3. Set brand colors:
   - Primary color: Match your app's primary color (e.g., #0f172a for dark, or your brand color)
   - Background color: Optional, typically white/light

4. Set accent color if desired (used for buttons, links)

5. Save changes

**Note:** Portal branding is shared across all Stripe-hosted pages (checkout, portal, invoices).
