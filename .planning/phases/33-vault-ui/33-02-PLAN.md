  <action>
    **Timeline API endpoint (`src/app/api/vault/timeline/route.ts`):**
    1. Create `GET` handler following the same auth pattern as `/api/sources/route.ts`
    2. Authenticate via `auth()`, return 401 if no session
    3. Query all statements for the user:
       ```sql
       SELECT id, sourceType, originalFilename, statementDate, transactionCount, pdfStoragePath, createdAt
       FROM statements
       WHERE userId = ?
       ORDER BY statementDate DESC NULLS LAST
       ```
       Use Drizzle: `db.select({ ... }).from(statements).where(eq(statements.userId, userId)).orderBy(desc(statements.statementDate))`
    4. Get transaction stats per statement — same pattern as `/api/sources/[sourceType]/statements/route.ts`:
       - Collect all statement IDs
       - Query transactions grouped by statementId with converted/skipped/pending counts
       - Build `statsMap: Map<string, { converted, skipped, pending }>`
    5. Map results to response shape:
       ```typescript
       {
         statements: Array<{
           id: string;
           sourceType: string;
           originalFilename: string;
           statementDate: string | null;  // ISO string or null
           transactionCount: number;
           hasPdf: boolean;  // derived from pdfStoragePath IS NOT NULL
           stats: { converted: number; skipped: number; pending: number };
         }>;
         totalSources: number;   // distinct sourceTypes
         totalStatements: number; // total count
         totalPdfs: number;       // count where hasPdf is true
       }
       ```
    6. IMPORTANT: serialize dates properly — `statementDate instanceof Date ? statementDate.toISOString() : null` (same pattern as sources route for Date serialization)
    7. Never expose `pdfStoragePath` to client — derive `hasPdf: pdfStoragePath !== null`
    8. Handle errors with try/catch, return 500 on failure

    **useVaultTimeline hook (`src/lib/hooks/use-vault-timeline.ts`):**
    1. "use client" directive
    2. Define query keys: `export const vaultKeys = { timeline: () => ["vault", "timeline"] as const }`
    3. Create `fetchVaultTimeline` function that calls `GET /api/vault/timeline`
    4. Export `useVaultTimeline` hook using `useQuery`:
       - queryKey: `vaultKeys.timeline()`
       - queryFn: `fetchVaultTimeline`
       - staleTime: `2 * 60 * 1000` (2 minutes, consistent with other hooks)
    5. Define and export the response type `VaultTimelineResponse` with the shape from step 5 above
    6. Also export individual `VaultTimelineStatement` type for the statement items
  </action>
