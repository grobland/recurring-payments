Create `src/components/features/feature-gate.tsx` with `"use client"` directive:

**FeatureGate Component:**

Props interface:
```typescript
interface FeatureGateProps {
  feature: Feature;
  children: React.ReactNode;
  fallback?: React.ReactNode;          // Custom locked state UI
  showUpgradePrompt?: boolean;          // Default true
  featurePreview?: React.ReactNode;     // Optional preview in modal
}
```

Implementation:
1. Use useState for access state and dialog open state
2. Use useEffect to call `getUserFeatureAccess(feature)` on mount
3. While loading (access is null), render fallback or null
4. If hasAccess is true, render children
5. If hasAccess is false:
   - If showUpgradePrompt is false, render fallback or null
   - Otherwise, render default locked placeholder (clickable dashed border card with Lock icon)
   - On click, open upgrade Dialog modal

**Upgrade Modal Content:**
- Title: "Upgrade to {tierName}" (from TIER_CONFIG)
- Description: "This feature requires the {tierName} tier"
- Feature preview section (if featurePreview prop provided)
- Feature list from TIER_CONFIG[requiredTier].features
- Footer with two buttons:
  - "Compare all tiers" (variant="outline") -> router.push("/pricing")
  - "Upgrade now" (variant="default") -> router.push("/settings/billing")

**LockedNavItem Component:**

Props:
```typescript
interface LockedNavItemProps {
  feature: Feature;
  children: React.ReactNode;
  className?: string;
}
```

Implementation:
1. Use useState/useEffect to check feature access
2. While loading, return null
3. If hasAccess, render children with className
4. If no access, render children with className + "opacity-50 cursor-not-allowed pointer-events-none"

Use existing patterns:

- Button from "@/components/ui/button"
- Lock icon from "lucide-react"
- useRouter from "next/navigation"
