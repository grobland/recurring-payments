
**Dark mode (.dark) — replace sidebar variables:**

```css
/* Warm charcoal instead of pure dark gray */
--sidebar: oklch(0.22 0.015 55);
/* Warm off-white text */
--sidebar-foreground: oklch(0.92 0.01 80);
/* Warm amber accent for emphasis */
--sidebar-primary: oklch(0.70 0.12 65);
--sidebar-primary-foreground: oklch(0.98 0.01 80);

--sidebar-ring: oklch(0.55 0.04 55);
```

These values add warmth by:
- Using hue angles in the 55-80 range (warm amber/tan/cream spectrum)
- Adding small chroma values (0.01-0.08) instead of 0 (achromatic)
- Light mode: cream background + peach accent
- Dark mode: warm charcoal background + amber accent
