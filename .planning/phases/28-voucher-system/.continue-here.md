<decisions_made>

### Voucher Management
- Stripe Dashboard only for creation (no custom creation UI)
- Both percentage-off and free-month discounts
- Single-use per user, always require expiration dates

### Trial Extensions
- Admin applies directly (user sees extended trial without action)
- Configurable duration per voucher
- Cumulative stacking allowed
- Trial users only (not for paid subscribers)

### Distribution
- Manual sharing (admin copies from Stripe)
- Checkout uses Stripe's native allow_promotion_codes field
- No automation triggers

### Admin Visibility
- Basic list of applied trial extensions in admin panel
- Show recipient, days added, date applied
- Rely on Stripe Dashboard for promo code usage stats
- No custom reporting needed
</decisions_made>
